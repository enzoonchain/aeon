# Data Drop Template

## Structure
```
[data point — the number]

[context — what it means]

[implication — why it matters]

this is base rn 🟦
```

## Rules
- Lead with the number
- Give context in one line
- State the implication clearly
- No speculation, just facts
- Max 186 chars
