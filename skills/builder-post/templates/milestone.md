# Milestone Post Template

## Structure
```
[what we shipped — 1 line]

[data point — 1-2 lines]

[why it matters — 1-2 lines]

this is base rn 🟦
```

## Rules
- Lead with what shipped, not how you feel about it
- Include at least one number
- Explain why it matters in one sentence
- No "we're excited" / "big news" / "🚀"
- Max 186 chars
