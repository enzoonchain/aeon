# Engagement Post Template

## Structure
```
[question — direct, specific]

[context — why you're asking]

this is base rn 🟦
```

## Rules
- Ask specific questions, not "what do you think?"
- Give context so people can answer meaningfully
- No "RT if you agree" / "like for part 2"
- Max 186 chars

## Example Outputs

### Example 1: Builder Question
```
question for builders:

what's the best use for unused DIEM?

→ agent compute
→ LP collateral
→ rental marketplace
→ something else?

this is base rn 🟦
```
