# Alpha Post Template

## Structure
```
[what we found — the insight]

[data — supporting numbers]

[implication — what it means for builders]

this is base rn 🟦
```

## Rules
- Lead with the insight, not the research process
- Support with data
- State what builders should do with this info
- No "we discovered" / "our research shows"
- Max 186 chars
