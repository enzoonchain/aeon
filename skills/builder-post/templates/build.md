# Build Post Template

## Structure
```
[what we're building — 1 line]

[why — 1-2 lines]

[status — where we are]

this is base rn 🟦
```

## Rules
- Show, don't tell
- Include status/progress
- Explain why it matters
- No "coming soon" / "stay tuned"
- Max 186 chars
